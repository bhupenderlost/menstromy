# Menstromy

Menstromy is a Node.js-based menstrual cycle tracking app.

## Features
- User authentication (JWT)
- Menstrual cycle tracking
- Cycle predictions
- Ecommerce Store

## Technologies
- Node.js, Express.js
- MongoDB
- JWT for authentication
- Firebase Admin SDK
- CORS for security

## Installation
```sh
git clone https://github.com/bhupenderlost/menstromy.git
cd menstromy
npm install
npm start
